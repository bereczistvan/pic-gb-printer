# PIC Game Boy Printer
Game Boy Printer emulator on a PIC16F18857 microcontroller

## Software
MPLAB X IDE v6.05

MPLAB XC8 Compiler v2.41
## Hardware
Microstick X2C+ Development board

PIC16F18857 microcontroller
